About Gabii

In the "About Gabii" section of this website, you can learn about Gabii, Italy and the excavations that have taken place thus far at the site.

Tomb 34

In the "Tomb 34" section, you will learn what a cappuccina grave is. You will also learn about the main grave of focus in this project - Tomb 34.

Other Graves Explored

In the Other Graves Explored section of this website, you will learn about the some of the other graves at Gabii and how they compare to Tomb 34.

Explore the Museum

In the "Explore the Museum" section of this website, you will be able to explore the interactive museum project created through Unity software. This interactive museum includes 3D models from Gabii. These modesl include Tomb 34 as well as the other graves explored in this project.
